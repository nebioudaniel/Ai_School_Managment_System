##  Overview
This is a comprehensive management platform designed to streamline school operations through **Artificial Intelligence**. Beyond standard student and staff tracking, this system leverages AI to provide predictive insights, automated scheduling, and personalized learning paths.

##  Key AI Features
* **Predictive Analytics:** Identifies students at risk of falling behind based on historical performance trends.
* **Automated Scheduling:** Uses optimization algorithms to generate conflict-free timetables for teachers and classrooms.
* **Smart Attendance:** Integration-ready for automated attendance marking and tracking.
* **AI Insights:** Generates automated reports for administrators to visualize school-wide growth.

## 📦 Installation
To get a local copy up and running, follow these steps:

1. **Clone the repository**
2. 
   git clone [https://github.com/nebioudaniel/Ai_School_Managment_System.git](https://github.com/nebioudaniel/Ai_School_Managment_System.git)



2. **Install dependencies**

npm install  # or pip install -r requirements.txt

3. **Set up environment variables**
4. 
Create a `.env` file in the root directory and add your credentials:

DATABASE_URL=your_db_url

AI_API_KEY=your_key_here


4. **Run the application**

npm run dev
